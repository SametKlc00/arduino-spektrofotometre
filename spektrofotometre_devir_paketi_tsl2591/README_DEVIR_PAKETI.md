# Arduino Tabanlı Düşük Maliyetli Spektrofotometre - Devir Paketi

Hazırlanma tarihi: 2026-06-22

Bu paket, spektrofotometre projesinin mezuniyet sonrası hocaya düzenli şekilde devredilmesi için hazırlanmıştır. Ana sürüm TSL2591 dijital ışık sensörü, Arduino, LED ışık kaynakları, küvet ve ışık sızdırmayan 3D kutu üzerine kuruludur.

## Klasör Yapısı

| Klasör | İçerik |
| --- | --- |
| 01_Kodlar | Arduino ve Python veri kayıt kodları |
| 02_Raporlar | Haftalık, vize, final, kutu farkı ve vişne suyu raporları için alan |
| 03_PDF_Ciktilar | Nihai PDF çıktıları için alan |
| 04_Gorseller | Devre, kutu, sensör, ölçüm ve ekran görüntüleri |
| 05_3D_Model_ve_Kutu | STL/3MF/CAD dosyaları ve baskı notları |
| 06_Donanim_Notlari | Parça listesi, satın alma ve bağlantı notları |
| 07_Kullanim_El_Kitapcigi | PDF, DOCX ve Markdown kullanım kılavuzu |


## Teslimden Önce

Eski raporlar, fotoğraflar, Arduino ekran görüntüleri, Python arayüz çıktıları ve 3D kutu dosyaları ilgili klasörlere eklenmelidir. Bu paket ana iskeleti ve güncel kullanım dokümanını hazır verir.
