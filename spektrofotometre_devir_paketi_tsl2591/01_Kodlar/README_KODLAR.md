# Kodlar

## tsl2591_olcum
TSL2591 sensörünün çalışıp çalışmadığını lux değeriyle test eder.

## tsl2591_absorbans_komutlu
Serial Monitor üzerinden `d`, `b`, `s` komutlarıyla dark, blank ve sample ölçümü alır. Absorbans hesabı örneğini içerir.

## python_serial_csv_logger
Arduino Serial Monitor kapalıyken ölçüm çıktısını CSV dosyasına kaydeder. `PORT` değeri bilgisayardaki COM portuna göre güncellenmelidir.
